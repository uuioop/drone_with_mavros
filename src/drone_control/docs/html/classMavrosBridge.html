Detailed Description

MAVROS桥接器类，封装了与MAVROS通信的功能，提供无人机控制接口

Note
mavros中本地位置指令是ENU坐标系，而不是NED坐标系
机体坐标系下的速度指令是FLU坐标系，而不是FRD坐标系
mavros使用的是ros的坐标系而非PX4的坐标系
