\doxysubsection{Detailed Description}
M\+A\+V\+R\+O\+S桥接器类，封装了与\+M\+A\+V\+R\+O\+S通信的功能，提供无人机控制接口 

\begin{DoxyNote}{Note}
mavros中本地位置指令是\+E\+N\+U坐标系，而不是\+N\+E\+D坐标系 

机体坐标系下的速度指令是\+F\+L\+U坐标系，而不是\+F\+R\+D坐标系 

mavros使用的是ros的坐标系而非\+P\+X4的坐标系 
\end{DoxyNote}
